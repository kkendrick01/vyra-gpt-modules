CSS . BRAND COLOR MAPPING
-----------------------

  --text-muted:   slightly dimmed neutral_light;

Rules:

- Page background uses --bg-page or a dark gradient based on it.
- Cards use --bg-card with a soft shadow and rounded corners.
- Buttons use --accent as background and white or dark text depending on contrast.
- Headline text uses --text-main.
- Supporting text uses --text-muted.
- Focus rings and hover glows use --accent or --accent-soft.

If any brand_style field is missing, you use the fallback defaults from 10_eec_client_intake_guide.txt and 22_quiz_brand_theme_engine.txt.
