Phase A: Pre-Flow

Objective: Remove every extraneous decision before the block starts.

  • One Block. One Objective.
    Write it at the top: "In the next 90 minutes I will: ____________________"
  • Load Inputs.
    Open the exact doc. Load source material, research, prior notes, examples, frameworks.
  • Environment Reset.
    Close Slack, email, DMs, feeds. Phone in another room or airplane mode. Only essential tabs open.
  • Micro Ritual.
    3 slow breaths. Say the rule out loud: "For the next 90 minutes, my only job is [objective]."
Why This Works: This pre-block deletes extraneous cognitive load. Your brain walks in clear. No decisions needed. Only execution.

Phase B: Deep Work

Objective: Stay inside flow conditions from minute 1 to minute 70.

Minute 0-5: Get Moving

Start ugly. No polishing. Put down a quick outline or bullet skeleton.

Minute 5-30: First Pass

Fill the outline. Use AI to assist, not decide.

"Score this hook 1-10 for curiosity and clarity."
"Highlight any lines that feel generic."
"Propose 3 stronger variations of this CTA that feel human."

Minute 30-45: Feedback Loop

Run hooks and CTAs through defined checks. Mark weak spots. Only edit pieces that change outcomes.

Minute 45-65: Refine for Outcome

Ask one question: "Does this piece move a real person closer to a real action?" Tighten structure. Clarify story. Sharpen proof.

If You Stall: Stand up 30 seconds. One sip of water. Re-read the objective. Begin again from the next line. Do not wander.

Phase C: Exit Ritual

Objective: Close the loop. Preserve momentum. Protect trust in the system.

  • Decide:
    Ship it. Queue it. Or save as "V1 Complete" with scheduled review time.
  • Tag and Store:
    Save in one labeled place: "Client / Date / Asset Type." No desktop clutter. No mystery files.
  • Score the Block:
    Rate focus, output quality, and emotional feel (1-5 each). Anything under 3 signals a system issue.
  • One Line Reflection:
    "Next block will be easier if I: _____________"
Why This Works: This closes cognitive loops. Your brain learns that starting a block leads to completion. That's how flow becomes repeatable, not rare.
