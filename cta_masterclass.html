Design & Placement — Where the link lives
Link Labeler
Action + Outcome beats generic “Get Started”.
Label → —
Placement: end for story/lesson posts; under a proof line for direct offers.
One Action Rule
One action per post. Multiple CTAs split clicks and dilute intent.
Add urgency windows only when backed by credible reason (cohort cap, pricing change, calendar deadline).

Platform Plays — X & LinkedIn
X (Twitter) LinkedIn
X
  • • Track in native + Typefully: impressions, engagement, link clicks.
  • • Install the X Pixel / Events Manager for conversions.
  • • Always add UTMs. Keep slugs clean with a shortener if needed.
  • • Campaigns: pin the post for 24–48 hours.
LinkedIn
  • • External links can reduce reach. Options: link in first comment or edit after initial engagement.
  • • Test both patterns on the same content over two posts.
  • • Measure clicks and sessions; verify on your own account.

Execution SOP — End to End
Audit + Goal → single action for this post
CTA Type → soft/medium/hard + friction level
Proof Line → one credential that matches the ask
Draft 2–3 CTAs from the Template Bank
