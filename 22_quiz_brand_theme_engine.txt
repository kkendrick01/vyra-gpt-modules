# **1. COLOR SYSTEM**

## **If onboarding includes explicit colors:**

Use:

* **primary_color** = the brand’s main color
* **secondary_color** = accent or contrasting color
* **accent_color** = highlight color
* **neutral_light** = light background / UI color
* **neutral_dark** = dark background / UI color

If fewer than 5 colors are provided, generate missing ones using rules below.

## **Color Application Rules**

Apply colors in this hierarchy:

* **Page background:** neutral_dark for dark themes, neutral_light for light themes
* **Card background:** slightly lighter/darker than page
* **Headlines:** primary_color
* **Subheadlines:** neutral_light or muted neutral
* **Buttons:** secondary_color (hover: darken by 12%)
* **Tags / labels:** accent_color on neutral_dark

### **Tone → Mode Rules**

* premium, executive, strategic → **dark mode**
* wellness, holistic, lifestyle → **light mode**
* creative → **dark with gradients**
* masculine fitness → **dark with strong accents**
* feminine coaching → **light with soft neutrals**

Fallback: **dark mode**.

# **2. TYPOGRAPHY ENGINE**

Infer by niche + tone.

### **Executive / Operators / Consultants**

* Headings: `"Inter Tight"` or `"Work Sans"`
* Body: `"Inter"`
* Bold headings, regular body

### **Wellness / Mindfulness**

* Headings: `"Lora"` or `"Playfair Display"`
* Body: `"Inter"`
* Softer line heights

### **Fitness**

* Headings: `"Oswald"` or `"Montserrat"`
* Body: `"Inter"`
* Tight spacing

### **Financial / Legal**

* Headings: `"Inter Tight"`
* Body: `"Inter"`

### **Creative**

* Headings: `"Sora"`
* Body: `"Inter"`

Fallback:
`system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

---

# **3. SPACING + LAYOUT SYSTEM**

Use an 8-point grid:

* 8px, 16px, 24px, 32px
* Section spacing: 48px–64px
* Cards: 24px–32px padding
* Form spacing: 16px vertical
* More spacing for authoritative tones

---

# **4. SHAPE LANGUAGE**

### **Executive / Strategic**

* Radius: 12px
* Pill buttons
* Smooth cards

### **Fitness**

* Radius: 6px
* Sharp buttons
* Minimal radius inputs

### **Wellness**

* Radius: 16px
* Soft, rounded theme

### **Creative**

* Dynamic radius (8–16px)

Fallback: **12px**.

---

# **5. MODE (LIGHT vs DARK)**

AUTO-SELECTION:

* “professional”, “executive”, “strategic”, “technical” → DARK
* “calm”, “wellness”, “holistic”, “mindfulness” → LIGHT
* “fitness”, “metabolic”, “performance” → DARK
* Missing tone → DARK

---

# **6. COMPONENT LOGIC**

### **Tags / Badges**

* Font: 12px
* Padding: 4px 12px
* Background: accent_color at 12% opacity
* Text: accent_color
* Radius: match shape language

### **Section Headers**

* Uppercase
* 1px tracking
* Muted color (neutral_light at 70%)

### **Inputs**

* Background: card background
* Border: 1px solid border-subtle
* Focus: 2px accent-color glow

### **Submit Button**

* Full width
* Background: secondary_color
* White text
* Hover: darken secondary_color by 12%

# **8. FALLBACK RULES**

If anything missing:

* Missing colors → generate via niche palette
* Missing tone → infer from niche
* Missing typography → use Inter for all
* Missing brand name → FirstName + “Method”
* Missing accent → saturate primary +25%
* Missing secondary → darken/lighten primary by 20%

Quiz must ALWAYS render with a complete theme.

# **10. LAYOUT + STRUCTURE RULES (MANDATORY)**

### **YOU MUST NEVER USE A TWO-COLUMN LAYOUT.**

No left/right split.
No sidebars.
No grid with two main columns.

Everything must be arranged in **ONE CENTERED COLUMN** with this exact order:

---

## **(1) HERO SECTION — TOP**

Must include, in this order:

1. **Brand pill**
   Example:
   “[Brand] · [Offer name]” or “[Brand] · [Signature Method]”

2. **Main headline (1–2 lines)**
   Focus on the diagnostic.
   Allowed patterns:

   * “Measure your [X] score in under 3 minutes.”
   * “See how aligned your [topic] really is.”
   * “Get a clear picture of your [pain point] today.”

3. **Subheadline (2–4 lines)**
   MUST explain:

   * Who this diagnostic is for
   * What insight it reveals
   * That it unlocks a **personalized 5-day plan based on the client's method**

   Required phrasing pattern:
   **“At the end, you will unlock a personalized 5 day email course built from [Brand/Method] principles, tailored to your answers.”**

4. **Optional 2–3 bullets about the diagnostic ONLY**
   Allowed:

   * What this quiz helps them see or measure
   * Why this matters
   * What blind spots it reveals

   NOT allowed:

   * Day-by-day descriptions
   * Course outline
   * Curriculum-style bullets

---

## **(2) QUIZ CARD — DIRECTLY UNDER HERO**

Single column.
Inline card.

Top of card:

* First name
* Best email

Then:

* 5–8 diagnostic questions
* Clear, branded labels
* Smart option pills or text areas

Button:

* Uses microcopy engine
* Can reference the 5-day plan ONLY in general terms, e.g.:

  * “Get my 5 day alignment plan →”
  * “Send me my personalized 5 day strategy →”

Under the button:

**ONE reassurance line**:

Example pattern:
“You will receive a 5 day email course tailored to your answers, starting in the next few minutes. No spam. Only targeted, practical guidance.”

---

## **(3) OPTIONAL TRUST / CREDIBILITY STRIP — BOTTOM**

Short section below quiz.
Same column.

Include:

* One small heading (“Built for [audience]” or “Why this diagnostic exists”)
* 2–3 short sentences about:

  * Credibility
  * Method
  * Professionalism

ABSOLUTELY NO:

* Day 0
* Day 1–5
* “What you’ll learn each day”
* Course outline of any form

---

## **STRICT PROHIBITIONS**

You must NEVER include:

* Day 1 / Day 2 / Day 3 on the landing page
* Any outline of the 5-day sequence
* Any mention of “this day covers…”
* Any curriculum bullets
* Any syllabus-like breakdown
* Any two-column layout

The diagnostic page must ALWAYS feel:

* Personalized
* Clean
* Modern
* Professional
* Fully brand-themed
* Focused ONLY on **insight now** + **promise of tailored plan after**
